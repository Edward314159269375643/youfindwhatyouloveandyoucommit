## 医院导诊小程序系统 - 快速开始指南

### 📋 系统概览

本系统包含两个主要部分：
1. **微信公众号聊天端** - 为患者提供智能导诊服务
2. **后台管理系统** - 供医院工作人员管理和分析数据

---

### 🚀 快速启动

#### Windows 用户
1. 双击运行 `init.bat` 初始化数据库
2. 双击运行 `start.bat` 启动服务器
3. 打开浏览器访问相应地址

#### 通用方法
1. 安装依赖：`npm install`
2. 初始化数据库：`npm run init-db`
3. 启动服务：`npm start`

---

### 🌐 访问地址

启动服务器后，可以通过以下地址访问：

| 页面 | 访问地址 |
|------|---------|
| **聊天端首页** | `http://localhost:3000/chat.html` |
| **管理后台登录** | `http://localhost:3000/admin/login.html` |
| **API 健康检查** | `http://localhost:3000/api/health` |

**提示**：系统会自动根据访问域名配置API地址，支持从其他设备访问。

### ⚙️ 配置说明

编辑 `.env` 文件配置数据库连接：

```env
PORT=3000
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=hospital_guidance
JWT_SECRET=your_jwt_secret_key
JWT_EXPIRES_IN=7d
```

### 🎯 常见问题

**Q: 数据库连接失败？**
A: 检查 `.env` 中的数据库配置，确保 MySQL 服务正在运行。

**Q: 无法上传图片？**
A: 确保 `uploads` 目录存在且有写入权限。

**Q: 登录失败？**
A: 运行 `npm run init-db` 初始化数据库和默认账户。

**Q: 从其他设备无法访问？**
A: 系统已支持跨设备访问，API地址会自动适配。确保防火墙允许3000端口访问。
