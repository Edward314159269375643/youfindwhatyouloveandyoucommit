
# 医院导诊小程序系统 - 快速部署

## 🎯 推荐方案：Railway（免费，更简单）

### 1. 打开 Railway
访问：https://railway.app/new

### 2. 连接 GitHub
- 点击 "Deploy from GitHub repo"
- 授权 GitHub 访问
- 选择你的仓库：`youfindwhatyouloveandyoucommit`

### 3. 配置环境变量
在 Railway 控制台的 "Variables" 标签添加：

| 变量名 | 值 |
|--------|-----|
| `JWT_SECRET` | `hospital-guidance-secret-key-2026` |
| `WECHAT_TOKEN` | `your-wechat-token` |
| `WECHAT_APP_ID` | `your-wechat-appid` |
| `WECHAT_APP_SECRET` | `your-wechat-appsecret` |

### 4. 部署
- 点击 "Deploy Now"
- 等待 1-2 分钟，部署完成后 Railway 会给你一个 HTTPS 地址

## 🎯 备选方案：Replit（超简单，免费）

### 1. 打开 Replit
访问：https://replit.com/new

### 2. 导入 GitHub 仓库
- 点击 "Import from GitHub"
- 粘贴你的仓库 URL：`https://github.com/Edward314159269375643/youfindwhatyouloveandyoucommit`

### 3. 配置运行命令
- 在 "Run" 配置中输入：`node server.js`

### 4. 添加环境变量
- 在 "Secrets" 中添加环境变量（同上）

### 5. 运行
- 点击 "Run"，Replit 会自动启动并给你一个 HTTPS 地址

---

## 🎯 备选方案：Render（免费）

### 1. 打开 Render
访问：https://dashboard.render.com

### 2. 创建 Web Service
- 点击 "New Web Service"
- 连接你的 GitHub 仓库
- 选择仓库并导入

### 3. 配置
- **Build Command**：留空或输入 `npm install`
- **Start Command**：`node server.js`
- **Branch**：`master`

### 4. 添加环境变量
在 "Environment" 标签添加变量（同上）

### 5. 部署
点击 "Create Web Service"
